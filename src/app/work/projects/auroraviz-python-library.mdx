## Overview

AuroraViz is a modern Python visualization library published globally on PyPI, designed to help data analysts and scientists create clean, professional visualizations with minimal effort. The library features curated color palettes optimized for both light and dark modes.

## Key Features

- **PyPI Publication**: Successfully published and maintained as a global Python package
- **Curated Color Palettes**: Professionally designed color schemes for various visualization needs
- **Light/Dark Mode Support**: Optimized palettes for both light and dark backgrounds
- **Easy Integration**: Simple API for quick adoption in existing Python workflows
- **Professional Aesthetics**: Focus on clean, publication-ready visualizations
- **Documentation**: Comprehensive documentation and usage examples

## Technologies Used

- **Python**: Core development language
- **Matplotlib**: Foundation library for visualization capabilities
- **PyPI**: Python Package Index for global distribution
- **Setup Tools**: Package configuration and distribution
- **GitHub**: Version control and issue tracking

## Challenges and Learnings

The main challenge was designing color palettes that work effectively across different backgrounds and use cases while maintaining accessibility standards. I learned about color theory, accessibility guidelines (WCAG), and the importance of comprehensive documentation for open-source projects. The PyPI publication process also taught me about package distribution and version management.

## Outcome

AuroraViz is now available globally on PyPI and can be installed with a simple pip command. The library has been adopted by data professionals looking for quick, professional visualization solutions. This project demonstrates my ability to contribute to the open-source community and create tools that solve real-world data visualization challenges.
